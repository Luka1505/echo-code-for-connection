ECHO is a private emotional reflection web platform built for the Code for Connection Hackathon.

# Features
- Mood check-ins with intensity tracking
- Private reflection journaling
- Emotional data visualisation dashboard
- Anonymous browser-based identity
- REST-based client-server architecture

# Tech Stack
Frontend: React (Vite), Tailwind CSS  
Backend: Flask (Python)  
Database: MongoDB
